# English ↔ Hebrew Speech Translator

🎤 Real-time speech-to-speech translation between English and Hebrew using Whisper, Groq, and Llama models.

---

## Features

- Real-time English ↔ Hebrew speech translation
- Speech-to-text transcription with Whisper Large V3 Turbo
- AI-powered translation using Llama 3.3 70B and Llama 4
- Optional high-accuracy reasoning mode for nuanced translations
- Audio playback for translated speech
- Lightweight Gradio interface
- Deployable on Hugging Face Spaces

---

## Tech Stack

- Python
- Gradio
- Groq API
- Whisper Large V3 Turbo
- Llama 3.3 70B
- Llama 4

## Local Setup

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/english-hebrew-speech-translator.git
cd english-hebrew-speech-translator
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Set up your environment variable:

```bash
GROQ_API_KEY=your_api_key_here
```

Run the application:

```bash
python app.py
```

---

## Setup

To run this locally or in your own Hugging Face Space:

1. Set up a `GROQ_API_KEY` in your environment variables or Space secrets
2. Install dependencies with:

```bash
pip install -r requirements.txt
```

3. Run the app:

```bash
python app.py
```
